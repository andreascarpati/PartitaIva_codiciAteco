
# Documenti Partita IVA – Agenzia delle Entrate

Repository strutturato per contenere i documenti ufficiali relativi alla mia Partita IVA e le fatture con dati sensibili oscurati.

## Struttura
- **agenzia/attività-principale/** → documento di attribuzione P.IVA
- **agenzia/attività-secondaria/** → aggiunta codice ATECO
- **fatture/** → fatture con nomi e dati personali cancellati
